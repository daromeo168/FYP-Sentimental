How our software works
1. The user needs to own a .csv file which can be converted from any tools such as Google Sheet, Microsoft Excel, Google Forms etc.
To convert Google Forms to Google Spreadsheets you can follow the instruction from this link
2. The header in the column of the .csv file as shown below in the highlight needs to be with the text of “TextDataReview” and the data is stored below that header in order to make our software recognize the format that we have set.
3. There is an button that says choose file, then you have to click on that button and find any .csv file from your device that has the same format as we have mentioned in (2) point.
4. After you have uploaded your file click on the “Back to homepage” button, then the website will redirect you to the homepage and you can see the result from our graphs which are located at the bottom of our homepage.
NOTE:
1. The result that you got after uploading the file will only see for temporary the result will not be stored in any kind of database or in your local drive, the data will be reset after you refresh the page.
2. Our website only works well with the 5 facilities that we have mentioned on the website, so the subject that is put in the survey for feedback/review should be anything that is related to those 5 facilities.
1. First we were collecting data from people using google survey
2. From there we stores those data into the google spread sheet
3. We query each data at a time from google spread sheet using our algorithm to evaluate the sentimental level of the data and convert it into a human readable form.
Using the algorithm to determine which one of the 5 facility that the data is belong to.
4. Finally we categorized all the data and display as info-graphic on the website
